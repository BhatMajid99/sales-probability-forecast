# 📊 Sales Probability Forecast

## 📌 Project Overview

This project aims to predict future product sales using basic probability concepts and past sales data. It uses Excel for storing data and Python for analysis and forecasting.

The project focuses on understanding sales trends, organizing datasets, and applying probability concepts for future prediction models.
